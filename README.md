# portfolio
this is a beginner-level portfolio website I created while learning HTML and CSS. The project showcases different sections such as Home, Services, Skills, Education, Experience, and Contact, all styled with basic CSS.
Key Features:                                                                                                                                                                        Responsive Design: The website is optimized for various screen sizes using CSS media queries.                                                                                            Navigation Menu: Simple navigation bar with links to different sections of the site.                                                                                                    Social Links: Icons linked to social media platforms (LinkedIn, Twitter, Instagram, and WhatsApp).                                                                                        Image Section: Includes a profile image and introductory text.
Technologies Used:                                                                                                                                                                        HTML5: For structuring the content.                                                                                                                                                           CSS3: For styling and layout design.
